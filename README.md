# Movie Recommendation System — CiaoDVD Dataset

**CSIT-557 | Final Project**

## Team Members

- Samuel Cordova
- Kareem Abdelrehiem

## Project Description

This project implements a movie recommendation system using the [CiaoDVD dataset](https://www.librec.net/datasets.html), a real-world dataset containing user ratings and trust relationships for DVD movies collected from the Ciao platform.

The system explores collaborative filtering and other recommendation techniques to generate personalized movie recommendations for users, leveraging both explicit rating data and social trust networks present in the dataset.
